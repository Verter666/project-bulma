# project-bulma

## Описание проекта
Учебный проект — адаптивный сайт, выполненный с использованием CSS-фреймворка **Bulma**. Сайт состоит из 8–10 страниц, объединенных единой тематикой, включая:

- Главную страницу
- О проекте
- Каталог / Сервисы / Продукты
- Галерею / Медиа
- Новости / Статьи / Блог
- Форма обратной связи / Подписка
- Контакты
- Дополнительные страницы (по необходимости)

Сайт полностью адаптивен и корректно отображается на мобильных устройствах, планшетах и десктопах.

---

## Используемые технологии
- HTML5
- CSS3
- CSS-фреймворк **Bulma**
- JavaScript (для интерактивных элементов и адаптивного меню)
- Git / GitHub для контроля версий
