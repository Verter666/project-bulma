FAQ

Ответы на частые вопросы о структуре, адаптивности и сдаче проекта.

Частые вопросы

Почему выбран Bulma?

Bulma дает готовую адаптивную сетку и набор компонентов без обязательного JavaScript, поэтому хорошо подходит для учебной верстки.

Можно ли открыть сайт без сервера?

Да. Это статический сайт, поэтому достаточно открыть файл index.html в браузере. Для деплоя подойдет GitHub Pages.

Что делает JavaScript?

Скрипт включает мобильное бургер-меню и показывает сообщение после демонстрационной отправки форм.

Что проверить перед сдачей?

Нужно пройтись по ссылкам, открыть страницы на разных ширинах экрана и убедиться, что формы проходят HTML-валидацию.

Перед сдачей

  • Проверить навигацию
  • Открыть мобильное меню
  • Проверить формы
  • Обновить README
